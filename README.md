## Навыки и знания
- **Типы тестирования**: функциональное, регрессионное, smoke, exploratory, usability  
- **Тест-дизайн**: классы эквивалентности, граничные значения, попарное тестирование, предугадывание ошибок, таблица принятия решений  
- **Документация**: тест-кейсы, чек-листы, баг-репорты, матрица трассировки  
- **Языки программирования**: Python, C# (базовый)  
- **Инструменты**: Chrome DevTools, Postman, Jira / YouTrack, GitHub  
- **Автоматизация тестирования**: базовые навыки написания автотестов на `pytest` и `NUnit`, работа с `Allure Reports` и `GitHub Actions CI`  
- **ОС и устройства**: Windows, Android (эмуляторы и реальные устройства)  
- **Языки**: Русский (родной), Английский (технический, A2 — читаю документацию и пишу баг-репорты)
